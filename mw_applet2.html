This page embeds an instrumented applet version of Molecular Workbench which supports logging of user interactions with model and scripted controls.

Try clicking the Open, Pause and Reset scripted buttons along with the model player controls then click the Generate Report button to see a detailed presentation of the data that can be saved.

This particular model simulating water flowing does not have any variables so the report consists only of user ActionItems.
